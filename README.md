# Entornos de Desarrollo - 08 SOLID, Inyección de Dependencias, Patrones, Arquitectura del Software y otros elementos avanzados
Tema 08. SOLID, Inyección de Dependencias, Patrones, Arquitectura del Software y otros elementos avanzados. Curso 2022/2023.

![imagen](https://raw.githubusercontent.com/joseluisgs/EntornosDesarrollo-00-2022-2023/master/images/entornos.png)

## Contenidos
1. SOLID.
2. Inyección de Dependencias.
3. Result y Railway Oriented Programming.
4. Delegados y Observadores.
5. Más Patrones de Diseño.
6. Arquitecturas del Software.
7. Reflexión y Anotaciones.
8. Compatibilidad Kotlin <-> Java.
